This seminar paper introduces EDLIoT, a novel scheduling algorithm designed to optimize energy consumption and reduce latency in Internet of Things (IoT) networks. The algorithm improves task distribution and resource allocation, resulting in up to 23% energy savings, 18% latency reduction, and a 25% increase in network lifetime. The paper also outlines future directions for smart network optimization.
